AI Powered Medical Waste Segregator

This project showcases an automated medical waste segregation system that combines machine learning, computer vision, and robotics to efficiently sort medical waste. Using a conveyor belt and a camera, the system captures real-time images of waste items, which are then classified into five categories by a Convolutional Neural Network (CNN) trained on a large dataset. The system employs image processing to create bounding boxes around the detected waste, and the coordinates are converted into G-code, guiding a robotic arm to pick up and place the waste into the correct bin. This solution reduces manual labor, enhances accuracy, and promotes safer, more efficient waste management in healthcare environments.
